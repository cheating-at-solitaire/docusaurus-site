# Printer Troubleshooting

If your printer is not working, follow the steps below.

## Printer Offline

1. Ensure the printer is powered on.
2. Check all cables are connected.
3. Verify the printer is connected to the network.
4. Restart the printer.

## Print Jobs Stuck

### Windows

1. Open **Settings**.
2. Select **Printers & Scanners**.
3. Choose your printer.
4. Open the print queue.
5. Cancel all print jobs.
6. Try printing again.

## Paper Jam

1. Switch off the printer.
2. Open the paper tray.
3. Carefully remove any jammed paper.
4. Close all covers.
5. Restart the printer.

Never pull paper forcefully.

## Poor Print Quality

Check:

- Ink or toner levels
- Paper quality
- Printer heads (run cleaning cycle)
- Correct paper type selected

## Cannot Find the Printer

Try:

- Restart your PC.
- Restart the printer.
- Check network connectivity.
- Disconnect and reconnect to Wi-Fi.
- Add the printer again through Windows Settings.

## Still Not Working?

Collect the following before contacting IT:

- Printer name
- Printer location
- Error message
- Screenshot (if possible)
- Whether other users can print
